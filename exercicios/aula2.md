## ⚡ Resumo dos exercícios

### Ficha A — Semáforo de Acesso ★
O utilizador introduz um código numérico (1, 2 ou 3) e o programa responde com o nível de acesso correspondente usando `if / elif / else`.

**Conceitos:** `if`, `elif`, `else`, `int(input())`, comparação com `==`

---

### Ficha B — Analisador de Passwords ★★
O programa analisa uma password introduzida pelo utilizador e classifica-a como Fraca, Média ou Forte com base no comprimento, presença de números e letras maiúsculas.

**Conceitos:** `len()`, `str.isdigit()`, `str.isupper()`, operadores `and` / `or`, `if / elif / else`

---

### Ficha C — Detetor de Phishing ★★★
O programa analisa o domínio de um endereço de email e classifica-o como legítimo, suspeito ou phishing com base em regras de domínio.

**Conceitos:** `str.endswith()`, `in`, operadores `and` / `or` / `not`, condições compostas, `if / elif / else`

## 💡 Dica antes de começar

Antes de escrever código, desenha o fluxograma no Draw.io:

1. Define o que o programa recebe (INPUT)
2. Mapeia as condições (losangos — decisões)
3. Define o que o programa mostra (OUTPUT)
4. Só então começa a escrever o `if / elif / else`
